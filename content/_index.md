# 欢迎来到我的博客

这是一个功能丰富的个人博客模板，基于 [Zola](https://www.getzola.org/) 静态站点生成器构建。

## 特性

- 📝 支持多种内容类型：文章、想法、诗歌、故事、翻译等
- 🏷️ 完整的标签和分类系统
- 🔍 可选的搜索功能（基于Meilisearch）
- 📱 响应式设计
- 🎵 音频播放器支持
- 🔄 自动化内容同步（从Apple备忘录）
- 🐘 Mastodon集成

## 快速开始

1. 克隆此仓库
2. 运行 `./init-template.sh` 初始化模板
3. 修改 `config.toml` 中的个人信息
4. 开始写作！

更多信息请查看 [README.md](README.md)。
