## 我的使用体验

这两周的使用体验是，CC 的好用其实不在于 coding 本身，而是搭建一套系统的、半自动化的开发流程。即使你完全不生成任何代码，也可以使用它来完成一些简单重复的任务，把它们完全自动化地集成到自己的开发流程中。

而且其实可以不只是用于 coding，也可以使用 Claude Code 来完成一套学习流程，这个我还在琢磨中。
